PPOD_Core

The core modules for the PPOD (Person-Project-Organization-Dataset) schema

URI: https://github.com/PPODschema/PPOD_Core#

Name: PPOD_Core

Classes

Class Description
Container A class holding the tree of all entities, needed for LinkML validation and transformation
Entity Root model class for all things and informational relationships, real or imagined.
        Annotation A bundle of properties that is used to provide additional information or context to a statement.
                ReferenceAnnotation An annotation providing source information about the annotated statement.
                RoleAnnotation An annotation giving details on the role an agent plays with respect to another agent, for instance characterizing an is_member_of relationship.
        NamedThing a databased entity or concept/class
                Agent A person, group, organization or project that provides a piece of information (i.e. a knowledge association)
                        Organization An agent corresponding to social institutions such as companies, societies, etc.
                        Person A person, whether alive, dead, real, or imaginary.
                        Project A class representing a project or collective endeavour of some kind
                InformationContentEntity a piece of information that typically describes some topic of discourse or is used as support.
                        Dataset A named collection of data, usually containing only one type of data
                        Publication Any "published" piece of information. Publications are considered broadly to include any document or document part made available in print or on the web - which may include scientific journal issues, individual articles, and books - as well as things like pre-prints, white papers, patents, drug labels, web pages, protocol documents, and even a part of a publication if of significant knowledge scope (e.g. a figure, figure legend, or section highlighted by NLP).
        Statement A reified RDF statement with slots for subject, predicate, and object as well as its evidence. It has similar semantics to Statements in Wikidata.
                AgentContactStatement A statement giving contact details for the particular agent.
                        AddressStatement A statement giving the address of the described agent.
                        EmailStatement A statement giving the email for the particular agent
                        PersonContactStatement A statement giving contact details for a person, in particular the parts of their name.
                                FirstNameStatement A statement giving the first name of the person.
                                FullNameStatement A statement giving the full name of the person.
                                LastNameStatement A statement giving the last name of the person.
                        PhoneNumberStatement A statement giving a phone number for the agent.
                        URLStatement A statement giving a URL for the agent, for instance an organizational or personal website.
                AgentToAgentStatement A statement describing the relationship between two agents.
                        EmployedByStatement A statement asserting that the agent who is the subject is employed by the agent who is the object.
                        EmploysStatement A statement asserting that the agent who is the subject employs the agent who is the object.
                        FundingProvidedViaStatement A statement asserting that the linked entity funds this particular resource.
                        HasMemberStatement A statement asserting that an agent has another entity as its member, for instance an organization having a person as its member.
                        HasParticipantStatement A statement describing a relation between a process and a continuant, in which the continuant is somehow involved in the process.
                        IsMemberOfStatement A statement asserting that an agent is a member of another entity, for instance a person being a member of an organization.
                        ParticipatesInStatement A statement describing a relation between a continuant and a process, in which the continuant is somehow involved in the process.
                        ProvidesFundingForStatement A statement asserting that the given agent provides funding for the associated agent.
                AgentToInformationContentStatement A statement describing the relationship from an agent to an information content entity.
                        ContributesToStatement A statement describing a resource that an entity makes contributions to.
                        CreatorOfStatement A statement asserting that the agent is the creator of a particular piece of information content
                HasPartStatement A statement asserting that a related resource that is included either physically or logically in the described resource.
                InformationContentDescriptionStatement A statement giving an attribute of a piece of information content, such as a document or dataset.
                        AccessRightsStatement A statement giving information about who can access the resource or an indication of its security status.
                        ContactPointStatement A statement giving relevant contact information for the cataloged resource. .
                        DatasetDescriptionStatement A statement giving an attribute of a dataset.
                                GeographicExtentStatement A statement giving the geographical area covered by the dataset.
                                SpatialResolutionStatement A statement giving the minimum spatial separation resolvable in a dataset, measured in meters.
                                TemporalCoverageStatement A statement giving the temporal period that the dataset covers.
                                TemporalResolutionStatement A statement giving the minimum time period resolvable in the dataset.
                        KeywordStatement A statement providing a keyword or tag describing the resource.
                        LandingPageStatement A statement giving a web page that can be navigated to to provide access to the resource.
                        PublisherStatement A statement describing the entity responsible for making the resource available.
                        TypeStatement A statement giving the nature or genre of the resource, for instance the data type.
                InformationContentToAgentStatement A statement describing the relationship from an information content entity to an agent.
                        ContributorStatement A statement identifying the entity responsible for making contributions to the resource.
                IsPartOfStatement A statement asserting entity is part of the related entity.
